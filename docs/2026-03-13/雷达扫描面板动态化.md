# 雷达扫描面板动态化交付

**日期**：2026-03-13  
**范围**：盯盘 Tab 顶部雷达面板 → 自动扫描动效 + 系统状态检测

## 变更摘要

### 问题
盯盘 Tab 顶部雷达 Hero 卡片为纯静态展示，硬编码 "LIVE" 徽章和 100% 进度条，无法反映系统真实扫描状态，视觉体验死板。

### 方案
1. **后端新增 API** `GET /v1/watchlist/scan-status` — 返回扫描系统实时状态（enabled、active_alerts、routes_scanned、status）
2. **前端提取组件** `RadarScanCard` — 独立组件处理雷达面板全部逻辑
3. **自动扫描动效** — 航线文本 2.5s 自动轮播、扫描线旋转、脉冲 + 圆环动画
4. **状态徽章动态化** — LIVE（绿） / PAUSED（黄） / OFFLINE（红） / IDLE（灰）
5. **进度条实时化** — 基于实际扫描覆盖率计算百分比
6. **统计行** — 实时展示扫描航线数、活跃监控数、系统状态
