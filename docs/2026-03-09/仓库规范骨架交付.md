## 3. 关键实现

### 3.1 Agent 角色落地

- [`architect.md`](agents/architect.md) 负责拆解任务卡、接口契约、验收标准与交接约束。
- [`code.md`](agents/code.md) 负责单功能实现、文档更新、`git commit` 与 `git push`。
- [`debug.md`](agents/debug.md) 负责故障复现、最小修复、回归验证与故障文档。

### 3.2 Skills 可复用手册

- [`skill-doc-and-commit.md`](skills/skill-doc-and-commit.md)：文档更新、提交、推送闭环。

### 3.3 Prompt 模板固化
