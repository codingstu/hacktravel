# 用户登录注册系统

**日期**: 2026-03-14  
**状态**: 已完成

## 功能概述

实现完整的用户认证系统，支持多种登录方式，匿名用户免费 3 次 AI 规划调用。

## 实现细节

### 后端模块

- **backend/app/models/auth.py** — 认证相关 Pydantic 模型
  - `RegisterRequest`: 注册请求（邮箱/手机二选一）
  - `LoginRequest`: 登录请求
  - `SocialLoginRequest`: 社交登录请求
  - `SendCodeRequest`: 短信验证码请求
  - `AuthUser`: 用户信息
  - `AuthResponse`: 认证响应

- **backend/app/services/auth_service.py** — Redis 存储的认证服务
  - 用户注册与登录
  - 短信验证码发送与验证（5 分钟有效期）
  - Token 管理（30 天有效期）
  - AI 使用次数追踪
  - Redis Key 设计：`hkt:auth:{type}:{id}`

- **backend/app/routes/auth.py** — REST API 端点
  - `POST /v1/auth/register` — 用户注册
  - `POST /v1/auth/login` — 邮箱/手机登录
  - `POST /v1/auth/social` — 社交账号登录
  - `POST /v1/auth/send-code` — 发送短信验证码
  - `GET /v1/auth/me` — 获取当前用户
  - `GET /v1/auth/ai-usage` — 检查 AI 使用次数
  - `POST /v1/auth/logout` — 退出登录

### 前端组件

- **frontend/services/types.ts** — 新增认证相关类型定义
- **frontend/services/api.ts** — 新增认证 API 调用函数
- **frontend/services/auth.tsx** — AuthContext 状态管理
  - Token 持久化（AsyncStorage）
  - 自动恢复登录状态
  - AI 使用次数限制检查
  
- **frontend/components/Avatar.tsx** — 自动生成头像
  - 根据用户名首字母生成
  - 支持中文字符提取
  - 12 色哈希映射
  
- **frontend/components/AuthScreen.tsx** — 登录注册弹窗
  - 邮箱 + 密码登录
  - 手机号 + 验证码登录
  - 17 个国家区号选择
  - Google/Facebook 社交登录
  - 注册流程 + 条款同意

### 集成改动

- **frontend/app/_layout.tsx** — 添加 AuthProvider
- **frontend/app/(tabs)/index.tsx** — AI 使用次数限制集成
  - 未登录用户最多 3 次免费 AI 规划
  - 超出次数自动弹出登录框
- **frontend/app/(tabs)/profile.tsx** — 退出登录适配
  - 使用 Avatar 组件显示头像
  - 集成 signOut 方法

## 核心规则

1. **匿名优先可用** — 不登录可使用核心功能
2. **3 次免费限制** — 匿名用户免费 3 次 AI 规划
3. **自动生成头像** — 根据用户名生成 Google/Telegram 风格头像
4. **多登录方式** — 支持邮箱、手机号、社交账号登录
